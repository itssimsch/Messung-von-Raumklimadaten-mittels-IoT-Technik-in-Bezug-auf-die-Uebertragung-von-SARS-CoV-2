◆ MH_PREHEAT_THRESHOLD

#define MH_PREHEAT_THRESHOLD   10

Defines a threshold for MHZ-19C-sensor values variation during preheating.

MHZ-19C-Preheating will only succeed once two sensor readings taken after another do not vary by more than this amount.

◆ loop()

void loop ( )

Predefined loop-function.

Is called repeatedly during runtime.

The AsyncElegantOTA.loop() statement must be called repeatedly throughout the loop function. Pausing the loop function directly by delay(LOOPDELAY) would cause unpredictable errors from the OTA-service, since the AsyncElegantOTA.loop() statement wouldn't be called for extended periods of time. Therefore, the loop is only paused for 1ms, and instead the counter-var 'timer' is introduced.
