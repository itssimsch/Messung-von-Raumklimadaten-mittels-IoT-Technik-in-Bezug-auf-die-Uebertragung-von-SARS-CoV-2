◆ loop()

void loop ( )

Predefined loop-function.

Is called repeatedly during runtime

The AsyncElegantOTA.loop() statement must be called repeatedly throughout the loop function. Pausing the loop function directly by delay(LOOPDELAY) would cause unpredictable errors from the OTA-service, hence the AsyncElegantOTA.loop() statement would be called in a very long time period. Therefore, the loop is only paused for 1ms, and instead the counter-var 'timer' is introduced.
